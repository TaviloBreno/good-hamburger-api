# 🍔 Good Hamburger API

API REST para gerenciamento de pedidos da lanchonete Good Hamburger com sistema inteligente de descontos.

## 🚀 Tecnologias

- **.NET 8.0** - Framework principal
- **ASP.NET Core** - API REST
- **Entity Framework Core** - ORM com InMemory Database
- **FluentValidation** - Validação de dados
- **xUnit + Moq** - Testes unitários
- **Swagger** - Documentação interativa

## 📋 Pré-requisitos

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- Git

## 🏃 Como executar

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/GoodHamburger.git
cd GoodHamburger

# Restaurar pacotes
dotnet restore

# Executar a API
cd GoodHamburger.API
dotnet run
